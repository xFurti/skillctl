# @skillctl/cli

See the [root README](../../README.md) and [docs site](https://xfurti.github.io/skillctl/) for full documentation.

Universal package manager for [Agent Skills](https://agentskills.io). v0.5.0 adds immutable remote locks, frozen restoration on new machines, scoped sync/prune, transactional project state, and stable JSON envelopes.
