# Leogriel brand assets

Leogriel's mark represents two woven threads: mint for discovery and synchronization, violet for verification and testing. The crossing strands form an abstract `L`/weave inside a stable hexagonal system.

## Files

- `leogriel-mark.svg` — primary scalable mark and source for square exports.
- `leogriel-logo.png` — 1024×1024 raster logo for profiles and package listings.
- `leogriel-banner.svg` — editable wide repository banner.
- `leogriel-banner.png` — raster README and social banner.
- `../docs/assets/logo.svg` — simplified documentation favicon/sidebar mark.
- `../docs/assets/social-preview.svg` and `.png` — 1280×720 Open Graph and Twitter Card image.

## Palette

| Role | Color |
|---|---|
| Deep navy | `#07111F` |
| Surface navy | `#10263D` |
| Mint | `#79F2C0` |
| Teal | `#20C7B5` |
| Violet | `#6D7CFF` |
| Lavender | `#B794F6` |
| Primary text | `#F4FCFF` |

Keep clear space around the mark and retain sufficient contrast. Do not recolor individual crossing segments in ways that obscure the woven structure.
